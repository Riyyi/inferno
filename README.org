* Inferno

Inferno game engine project.

* Download

** Clone

#+BEGIN_SRC sh
$ git clone https://github.com/riyyi/inferno
$ cd inferno
$ git submodule update --init --recursive
#+END_SRC
Or
#+BEGIN_SRC sh
$ git clone --recursive https://github.com/riyyi/inferno
#+END_SRC

** Update

#+BEGIN_SRC sh
$ git pull
$ git submodule update --recursive
#+END_SRC

* Build instructions

#+BEGIN_SRC sh
$ mkdir build
$ cd build
$ cmake .. && make
#+END_SRC
